### **Useful SQL Functions**

|Function|Description|Example|
|---|---|---|
|`COUNT()`|Number of rows|`SELECT COUNT(*) FROM users;`|
|`SUM()`|Total of a numeric column|`SELECT SUM(price) FROM orders;`|
|`AVG()`|Average value|`SELECT AVG(age) FROM users;`|
|`MAX()` / `MIN()`|Highest / lowest value|`SELECT MAX(score) FROM tests;`|
