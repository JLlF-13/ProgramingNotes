### Common SQL Data Types

When creating tables in SQL, you need to define the **type of data** each column will store. Here are the most common ones:

| Data Type    | Description                               | Example Value           |
| ------------ | ----------------------------------------- | ----------------------- |
| `INT`        | Integer number                            | `42`                    |
| `VARCHAR(n)` | Variable-length string (max n characters) | `'hello'`               |
| `CHAR(n)`    | Fixed-length string                       | `'yes '` (padded)       |
| `TEXT`       | Long text (no length limit)               | `'This is a paragraph'` |
| `DATE`       | Calendar date                             | `'2025-09-06'`          |
| `BOOLEAN`    | True or false                             | `TRUE`, `FALSE`         |
| `FLOAT`      | Decimal number                            | `3.14`                  |
