A lightweight, fast, and highly customizable editor developed by Microsoft. It supports many programming languages and has a rich ecosystem of extensions. I use it mainly for web development and scripting.


---

### Recommended Plugins

- **Prettier – Code Formatter** Automatically formats your code to keep it clean and consistent.
    
- **Live Server** Launches a local development server with live reload for static and dynamic pages.
    
- **GitLens** Enhances Git capabilities inside VS Code, showing commit history, blame annotations, and more.
    
- **Material Icon Theme** Adds colorful and intuitive icons to your file explorer.
    
- **Error Lens** Highlights errors and warnings directly in your code, making them more visible.
    
- **JSON Crack** Visualizes JSON data as interactive graphs—great for understanding complex structures.
    
- **Live Share** Enables real-time collaboration with other developers directly inside VS Code.
    
- **SQLite Viewer** Lets you open `.sqlite` or `.db` files directly in VS Code, browse tables, run queries, and view results.
	


> Most programming languages have their own dedicated plugin—just search for the one you need!
