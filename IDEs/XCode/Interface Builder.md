Interface Builder is Xcode’s visual tool for designing UIKit‑based interfaces. It allows you to create screens using drag‑and‑drop components without writing layout code manually.

---

### Key Elements

#### **1. Storyboards & XIBs**

- **Storyboards**: Multiple screens in a single file
    
- **XIBs**: Individual reusable views
    
- Both support Auto Layout and constraints
    

#### **2. Auto Layout**

Defines how views resize and reposition:

- Constraints (leading, trailing, width, height…)
    
- Safe areas
    
- Intrinsic content size
    

#### **3. Inspectors**

Right‑side panels for configuration:

- Attributes Inspector
    
- Size Inspector
    
- Connections Inspector
    

#### **4. Outlets & Actions**

Connect UI elements to Swift code:

- `@IBOutlet` → references to UI elements
    
- `@IBAction` → triggers for user interactions
    

#### **5. Preview Assistant**

Shows a live preview of your storyboard on different devices.
