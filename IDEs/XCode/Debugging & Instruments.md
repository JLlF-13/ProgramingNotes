Xcode provides powerful tools to debug, analyze, and optimize your app.

---
### Debugging Tools

#### **1. Breakpoints**

Pause execution to inspect:

- Variables
    
- Call stack
    
- Memory state
    

#### **2. LLDB Console**

Run commands to inspect or modify runtime values.

#### **3. View Debugger**

Visualize your UI hierarchy in 3D:

- Detect misplaced constraints
    
- Inspect view frames and layers
    

#### **4. Logs & Console**

Track:

- Print statements
    
- Errors
    
- Network logs
    

---
### Instruments

Instruments is a performance analysis suite included with Xcode.

Common instruments:

- **Time Profiler** – CPU usage
    
- **Leaks** – Memory leaks
    
- **Allocations** – Memory footprint
    
- **Energy Log** – Battery impact
    
- **Network** – Traffic monitoring
    

Instruments helps you identify bottlenecks, optimize performance, and ensure smooth user experiences.
