**Electron** is a framework that allows you to build cross-platform desktop applications using **web technologies** like HTML, CSS, and JavaScript. It combines **Chromium** (for rendering) and **Node.js** (for backend logic), making it possible to create powerful apps with a single codebase.

### Requirements

Before installing Electron, you need:

- **Node.js installed** on your system
    
- An existing project using **any web-based technology** (e.g., plain JavaScript, React, Vue, etc.)
### Installation

Once your project is set up, you can install Electron using npm:

```bash
npm install electron --save-dev
```

Electron will look for the `main` field in your `package.json` to start the app.
