Tables in Markdown are used to organize data in rows and columns. They're great for showing comparisons, lists, or structured information.

- The **first row** defines the headers.
    
- The **second row** (with dashes) separates headers from content.
    
- You can align text using colons:
    
    - `:---` → left-aligned
        
    - `:---:` → center-aligned
        
    - `---:` → right-aligned

```
| Manga     | Author | Volumes       |
|----------|-----|------------|
| Bleach    | Tite Kubo  | 74  |
| One piece      | Eiichiro Oda  | ???     |
```
