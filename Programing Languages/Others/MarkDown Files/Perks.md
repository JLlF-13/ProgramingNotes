Markdown is simple, but with a few extra tricks, you can make your notes cleaner, more expressive, and easier to navigate.

#### Horizontal Separators

Use three dashes, asterisks, or underscores to create a visual break:

```
---
***
___
```

---
***
___

### Decorative Lines

```
###### ────────────────
```

###### ────────────────

#### Use triple backticks to format code:
```
This is a backtick --> `
```

You can also specify the programming language after the first set of backticks to enable syntax highlighting:
```python
print("Python")
```

#### Inline Code
Use single backticks for inline code:

```
print("Single backticks")`
```

Result:
	`print("Single backticks")`

#### Hidden Comments (Obsidian only)

Add comments that won’t render in preview:

```
%% This is a hidden comment %%
```

%% This is a hidden comment %%
