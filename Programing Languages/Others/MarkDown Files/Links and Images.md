There are 2 types of link internal link and external links

- Internal link in Obsidian like this: `[[File Name]]`

If you have some files that are the same name but are conected to diferent files like in this note do this: 

`[[File Route | File Custom Show Name]]`
`[[Programing Languages/VERSE/CODE/Print|Print]]` --> Print
`[[File Route]]` --> Programing Languages/VERSE/CODE/Print


- External link: `[My GitHub](https://github.com/JLlF-13)` --> [My GitHub](https://github.com/JLlF-13)
