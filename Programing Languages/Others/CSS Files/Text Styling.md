Text styling in CSS allows you to control how text appears on your webpage—its alignment, font, size, weight, spacing, and decorative effects. These properties help improve readability and visual hierarchy.

### **Text Alignment**

Controls the horizontal alignment of text.

```css
text-align: left;     /* Default */
text-align: center;
text-align: right;
text-align: justify;
```


---

### **Font Family**

Defines the typeface used for text.

```css
font-family: Arial, sans-serif;
font-family: 'Roboto', sans-serif;
```

You can use web-safe fonts or import custom fonts using `@font-face` or Google Fonts.

### **Font Size**

Sets the size of the text.

```css
font-size: 16px;
font-size: 1.2em;
font-size: 120%;
```

Use `em` or `rem` for scalable and responsive design.

### **Font Weight**

Controls the thickness of the text.

```css
font-weight: normal;
font-weight: bold;
font-weight: 300; /* Light */
font-weight: 700; /* Heavy */
```


---

### **Text Decoration**

Adds styling like underlines or strikethroughs.

```css
text-decoration: underline;
text-decoration: line-through;
text-decoration: none;
```

### **Text Transform**

Changes the case of text.

```css
text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;
```

### **Letter & Word Spacing**

Adjusts spacing between characters or words.

```css
letter-spacing: 2px;
word-spacing: 10px;
```

### **Line Height**

Controls vertical spacing between lines of text.

```css
line-height: 1.5;
```

Improves readability, especially for paragraphs.

### **Text Shadow**

Adds shadow effects to text.

```css
text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
```

Useful for creating depth or emphasis.
