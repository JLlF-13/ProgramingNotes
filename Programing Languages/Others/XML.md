Also known as Extensible Markup Language

- Used to store and transport structured data.
    
- Tags are customizable and must follow strict syntax rules.
    
- Commonly used in configuration files, data exchange between systems, and APIs.

---

**Additionally, here are 3 key things to know about XML:**

- Tags can have attributes, for example: `<tag hi="attribute"/>`
    
- All tags must have a start and end: `<tag>content</tag>` or be self-closing: `<tag/>`
    
- Tag names cannot contain spaces.


```xml
<?xml version="1.0" encoding="UTF-8"?>
<mangas>
  <manga id="1">
    <title>Blue Lock</title>
    <genre>Sports, Action</genre>
    <sport>Football</sport>
    <author>Muneyuki Kaneshiro</author>
    <year>2018</year>
  </manga>
  <manga id="2">
    <title>Kuroko no Basket</title>
    <genre>Sports, Drama</genre>
    <sport>Basketball</sport>
    <author>Tadatoshi Fujimaki</author>
    <year>2008</year>
  </manga>
</mangas>
```
