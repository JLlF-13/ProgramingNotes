Markdown is a **lightweight markup language** used to format plain text documents with simple syntax.

- Commonly used in **README files**, **documentation**, **wikis**, and **note-taking apps**.
    
- Converts easily to **HTML** for web display.
    
- Markdown is widely supported—even apps like **Discord**, **Notion**, and **GitHub** use it behind the scenes.
    

Right now, you're reading a Markdown document! It’s designed to be easy to write and even easier to read.

Dont worry markdown is very simple
