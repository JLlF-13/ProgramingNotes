HTML provides **three main types of lists** to organize content:

#### 1. **Ordered List** (`<ol>`)

- Displays items in a numbered sequence.
    
- Useful when order matters (e.g., steps in a process).

```html
<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Have breakfast</li>
</ol>
```

#### 2. **Unordered List** (`<ul>`)

- Displays items with bullet points.
    
- Used when order is not important.

```html
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>
```

#### 3. **Description List** (`<dl>`)

- Used for terms and their definitions or descriptions.
    
- Contains `<dt>` (term) and `<dd>` (description).

```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
```

_Tip:_ You can nest lists inside each other to create sublists, and customize list styles with CSS.
