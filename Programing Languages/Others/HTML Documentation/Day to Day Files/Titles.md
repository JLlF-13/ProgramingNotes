In HTML, you create titles and section headings using the `<h1>` to `<h6>` tags. These tags define the **hierarchy and importance** of your content:

- `<h1>` is the **main title** of the page — usually used once.
    
- `<h2>` to `<h6>` are used for **subheadings**, with decreasing importance.

Each level helps structure your content for both readers and search engines.

```html
<h1>Hi</h1> <!-- Main heading -->
<h2>Hi</h2> <!-- Subheading -->
<h3>Hi</h3> <!-- Sub-subheading -->
<h4>Hi</h4>
<h5>Hi</h5>
<h6>Hi</h6>
```

The browser will display each heading in a different size, with `<h1>` being the largest and `<h6>` the smallest.
