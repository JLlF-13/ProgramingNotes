### Simple comment

>PHP supports C, C++ and Unix shell-style comments

```PHP
<?php  
echo "How are you?"; // This is a one-line c++ style comment   
#echo "Fine Thanks"; This is a one-line shell-style comment  
?>
```

### Multi-line Comment

PHP also supports Multi-Line comments 

```php
<?php  
echo "How are you?"; 
/* 
echo "I not fine";
echo "Silk Song"
*/  
echo "Fine Thanks"; 
?>
```

### One Line Comments

PHP also supports One Line comments 

```php
<h1>This is an <?php # echo 'simple';?> example</h1>  
<p>The header above will say 'This is an example'.</p>
```
