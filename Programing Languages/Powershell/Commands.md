| Command          | Windows CMD | Description                                       | PowerShell Example                                                                          | CMD Example                           |
| ---------------- | ----------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------- | ------------------------------------- |
| Get-Location     | cd          | Shows the current working directory               | Get-Location                                                                                | cd                                    |
| Get-ChilItem     | dir         | Lists files and directories                       | Get-ChildItem -Force                                                                        | dir /a                                |
| Set-location     | cd          | Changes Directory                                 | Set-Location C:\Users                                                                       | cd C:\Users                           |
| New-Item         | mkdir/echo  | Creates a new directory or file                   | New-Item -Name "notes.txt" -ItemType File                                                   | echo. > notes.txt \| mkdir new_folder |
| Remove-Item      | del/rmdir   | Deletes files or directories                      | Remove-Item notes.txt                                                                       | del notes.txt \| rmdir folder /s      |
| Copy-Item        | copy        | Copies files or directories                       | Copy-Item notes.txt backup.txt                                                              | copy notes.txt backup.txt             |
| Move-Item        | move        | Moves or renames files                            | Move-Item old.txt new.txt                                                                   | move old.txt new.txt                  |
| Write-Output     | echo        | Prints text to the terminal                       | Write-Output "Hello world"                                                                  | echo Hello world                      |
| Get-Content      | type        | Displays file content                             | Get-Content notes.txt                                                                       | types notes.txt                       |
| Select-String    | findstr     | Searches text using patterns                      | Select-String "error" log.txt                                                               | findstr "error" log.txt               |
| Get-Process      | tasklist    | Displays running processes                        | Get-Process                                                                                 | tasklist                              |
| Stop-Process     | taskkill    | Terminates a process                              | Stop-Process -Id 1234                                                                       | taskkill /PID 1234                    |
| Get-Help         | help        | Opens the help page for a command                 | Get-Help Get-ChildItem                                                                      | help dir                              |
| Test-Connection  | ping        | Sends ICMP packets to test if a host is reachable | Test-Connection google.com                                                                  | ping google.com                       |
| Get-NetIPAddress | ipconfig    | Displays network interface information            | Get-NetIPAddress                                                                            | ipconfig                              |
| whoami           | whoami      | Shows the current user                            | whoami                                                                                      | whoami                                |
| New-LocalUser    | net user    | Creates a new local user                          | New-LocalUser -Name "newuser"                                                               | net user newuser /add                 |
| Set-User         | net user    | Sets or changes a user's password                 | Set-LocalUser -Name "newuser" -Password (ConvertTo-SecureString "1234" -AsPlainText -Force) | net user newuser 1234                 |
