
### Scanner

`Scanner`: Used to store inputed data by the user

```java
Scanner scanner = new Scanner(System.in);
System.out.print("Give me a name: ");
String name = scanner.nextLine();

println(name)
```

Output:

```
Select your Option: Joan

Joan
```

Line by line:

`Scanner scanner = new Scanner(System.in);`: Used to initialize the scanner
`System.out.print("Give me a name: ");`:  Shows message to make it more user friendly
`String name = scanner.nextInt();`: Actual input

`printl(name)`: Shows input
