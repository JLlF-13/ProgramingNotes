In java an import is done like this 

```java
import pacakge.module.ClassName;
```

```java
import java.io.Scanner;
```

You can also import everything from a module with a `*` after the module --> `import java.io.*`

### These are some of the most important and usefull packages

| Package             | Purpose / Usage                                           |
| ------------------- | --------------------------------------------------------- |
| [[java.lang]]       | Core classes like `String`, `Math`, `Object`, `System`    |
| [[java.util]]       | Data structures and utilities like `ArrayList`, `HashMap` |
| [[java.io]]         | Input/output operations, file handling                    |
| `java.nio`          | Advanced I/O operations, buffers, channels                |
| `java.net`          | Networking capabilities (sockets, URLs)                   |
| `java.sql`          | Database connectivity using JDBC                          |
| `javax.swing`       | GUI components like buttons, windows, panels              |
| `java.time`         | Date and time API (e.g., `LocalDate`, `Duration`)         |
| [[java.xml]]        | Used for reading and creating XML files                   |
| org.jdom2           | Used for reading and creating XML files                   |
| [[com.google.gson]] | Used for reading and creating JSON files                  |
| [[java.sql]]        | Used to interact with Databases                           |
