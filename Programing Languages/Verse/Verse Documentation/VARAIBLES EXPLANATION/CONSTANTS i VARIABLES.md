### CONSTANTS

Les constants son variables que una vegada definides i inicialitzat el programa no es poden canviar, aquesta nomes es pots canviar desde el propi codi

```verse
name : string = "Joan"
```

###  VARIABLES
Les variables com el seu propi nom ja ho diu poden variar enmig del programa via el parametre ```set```  

```verse
var age : int = 19 
```

Aqui veim com podem canviar el valor

```verse
set age = 20
```
