#### Basic commands to initialize and configure your Git environment.

Start a new repo in git
```
git init
```

Clon a repo
```
git clone <repo-url>
```

## Initial configuration (important)

Set username:
```
git config --global user.name "<your-name>"
```

Set email:
```
git config --global user.email "<your-email>"
```

Check configuration:
```
git config --list
```
