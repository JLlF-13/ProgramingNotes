#### Essential commands for everyday coding, staging, and committing.

Shows the states of the changes
```
git status
```

Add changes to the preparation area
```
git add <file>
```

Confimr the changes with a message
```
git commit -m <message>
```

Show the commit history:
```
git log
```

Restore a file to the last committed state:
```
git restore <file>
```

Remove a file from the repository:
```
git rm <file>
```
