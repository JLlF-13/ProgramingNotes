#### Powerful tools for debugging, optimizing, and managing complex workflows.

Compares 2 commits, branches or files
```
git diff <a> <b>
```

Shows the datalis of a  commit
```
git show <hash>
```

Saves temproary changes and cleens the working tree
```
git stash
```

Restores the changes made in the stash
```
git stash pop
```

Apply a spicific commit in the current branch
```
git cherry-pick <hash>
```

Reapply commits on another bases
```
git rebase <base>
```
